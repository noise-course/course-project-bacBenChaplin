This folder should contain the following files to run the notebook:

1. `100-packet-traffic.pcapng` - Original dataset from nPrint leaderboard
2. `clean-traffic.pcap` - Generated by the notebook (or provided)
3. `print.npt` - Generated by running nPrint (or provided)

Only the first one is necessary if you are going to run the notebook from the beginning.

Please find the files on the [Google Doc](https://drive.google.com/drive/folders/1vXRKuW0KuB50WY1vgi5GFwIn-coFk_25?usp=sharing)
